Về trang này

Trang phân tích 16 đội tham dự The International 2026, dùng chỉ số thi đấu 3 tháng gần nhất lấy từ DLTV.org.

Cập nhật tự động: một GitHub Actions workflow chạy mỗi ngày, cào lại dữ liệu và commit vào repo; GitHub Pages tự phục vụ bản mới. Bạn cũng có thể chạy node scraper/scrape.mjs để cập nhật thủ công.

Các chỉ số: Winrate, Kills/Deaths/Assists trung bình mỗi trận, chênh lệch K–D, First Blood %, F10 (giành 10 kill đầu), tỉ lệ thắng khi có First Blood / F10, và thời lượng trận trung bình.

Ghi chú: một số đội có thể chưa hiện dữ liệu nếu slug DLTV chưa được xác minh — xem README để chỉnh scraper/teams.config.json.
